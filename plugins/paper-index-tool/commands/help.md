Display help information for paper-index-tool CLI commands.

## Usage

```bash
# Show general help
paper-index-tool --help

# Show command-specific help
paper-index-tool COMMAND --help

# Show version
paper-index-tool --version
```

## Arguments

- `COMMAND` (optional): Specific command to get help for
- `--help` / `-h`: Show help information
- `--version` / `-v`: Show version information

## Examples

```bash
# General help
paper-index-tool --help

# Command help
paper-index-tool search --help

# Version information
paper-index-tool --version
```

## Output

Displays usage information, available commands, and options.
